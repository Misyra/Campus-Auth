# 校园网认证助手 使用说明

## 快速开始

### 启动

**Windows 用户**

双击运行 `start.exe`，程序会自动下载运行环境并启动 Web 控制台。如果没有自动打开浏览器，请手动访问 `http://127.0.0.1:50721`，或点击右下角托盘图标。

> **📌 更新程序**：双击运行 `update.exe` 即可将代码更新到最新版本，更新后重新运行 `start.exe`。首次运行会自动下载 Git（仅 Windows，需联网）。

**macOS / Linux 用户**

在终端进入项目目录，执行：
```bash
chmod +x start.sh
./start.sh
```

> macOS/Linux 版本可能存在兼容性问题，遇到问题欢迎提交 issue 反馈。

**macOS / Linux 更新代码**

手动使用 Git 拉取最新代码：
```bash
git remote add origin https://github.com/Misyra/Campus-Auth.git  # 首次需添加远程仓库
git pull origin main
```

**资源占用**

- 后台常驻约 50 MB 内存，登录时短暂升高
- 关闭窗口即结束程序，如需长时间监控请开启"开机自启动"
- 仅需开机认证一次的用户，建议开启"自动登录，成功后退出"

---

### 首次配置

首次启动会弹出引导向导，按提示填写：

1. **账号** — 学号、密码、认证地址
2. **运营商** — 移动/联通/电信/无
3. **监控** — 检测间隔、暂停时段、开机自启
4. **确认** — 检查无误后完成

### 日常使用

启动后点击 **启动监控** 即可。程序自动检测网络状态，断网时自动登录，恢复后继续监控。

## 界面说明

左侧导航栏包含以下页面：

| 页面 | 用途 |
|------|------|
| 仪表盘 | 运行状态、快捷操作、实时日志 |
| 设置 | 账号、监控、浏览器等配置 |
| 配置方案 | 为不同网络环境创建独立配置 |
| 任务管理 | 管理浏览器登录任务 |
| Python 脚本 | 脚本直接登录（无需浏览器） |
| 日志 | 查看历史日志 |
| 外观 | 自定义主题、背景 |

---

## 仪表盘

- **统计卡片** — 运行时长、检测次数、失败计数、最后检测
- **状态横幅** — 显示当前网络状态（已连接/已断开）
- **快捷操作** — 手动登录、网络测试、开机自启开关
- **实时日志** — 支持按级别筛选、关键词搜索、截图预览

## 设置

### 账号

| 项目 | 说明 |
|------|------|
| 用户名 | 学号或认证账号 |
| 密码 | 认证密码 |
| 认证地址 | 校园网认证页面 URL |
| 运营商 | 移动/联通/电信/无 |

### 网络与监控

| 项目 | 说明 |
|------|------|
| 检测间隔 | 多久检测一次网络，默认 5 分钟 |
| 暂停时段 | 设置暂停时间段（如凌晨 0-6 点） |
| 代理 | HTTP 代理地址，留空不使用 |

### 系统

| 项目 | 说明 |
|------|------|
| 日志级别 | 一般选 INFO 即可 |
| 日志保留天数 | 超过天数自动清理 |
| 启动后自动监控 | 开启后无需手动点击 |
| 最小化到托盘 | 启动后最小化运行 |
| 自动登录后退出 | 登录成功后自动退出程序 |
| 开机自启动 | 开机自动运行 |

### 浏览器

| 项目 | 说明 |
|------|------|
| 无头模式 | 隐藏浏览器窗口，推荐开启 |
| 低资源模式 | 禁止加载图片，减少内存占用 |
| 页面超时 | 加载等待时间，网络慢可调大（如 30000ms） |

## 任务管理

程序内置默认任务，一般无需修改。如需自定义：

- **使用** — 设为当前活动任务
- **编辑** — 修改任务配置
- **调试** — 逐步执行，查看截图和结果
- **导入/导出** — 分享任务配置

还可通过"远程仓库导入"从 [Campus-Auth 任务仓库](https://github.com/Misyra/campus-auth-tasks) 一键导入适配方案。

---

## 配置方案

适用于**多个网络环境**的用户（如宿舍、教学楼使用不同网关）。

1. **创建方案** — 为不同网络分别创建
2. **设置匹配规则** — 填写网关 IP 或 WiFi 名称
3. **开启自动切换** — 连接新网络时自动应用

点击"检测当前网络"可自动获取网关 IP 和 WiFi 名称。

## 常见问题

**Q: 启动后一直显示"网络已断开"**
检查认证地址和账号密码，点击"手动登录"测试一次，查看日志错误信息。

**Q: 登录超时**
调大"页面超时"值（如 30000ms），或关闭"低资源模式"。

**Q: 检测间隔设多少合适**
一般 3-5 分钟。太短会频繁启动浏览器，太长则恢复较慢。

**Q: 多个网络怎么配置**
在"配置方案"页面为每个网络创建方案，填写网关 IP 或 WiFi 名称，开启自动切换。

**Q: 如何分享任务**
本地分享：点击"导出"下载 JSON 文件，同学导入即可。
社区分享：提交到 [Campus-Auth 任务仓库](https://github.com/Misyra/campus-auth-tasks)。

**Q: 内存占用太大**
开启"无头模式"和"低资源模式"，或开启"自动登录后退出"。
