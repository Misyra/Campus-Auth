JSON 配置说明

支持的步骤类型

  • navigate - 打开网页
  • input - 输入文本
  • click - 点击元素
  • select - 选择下拉框
  • wait - 等待元素出现
  • eval - 执行JavaScript

可用变量

  • {{username}} - 用户名
  • {{password}} - 密码
  • {{isp}} - 运营商后缀
  • {{url}} - 认证地址

示例

{
"steps": [
  {
    "type": "input",
    "selector": "#username",
    "value": "{{username}}{{isp}}",
    "description": "输入用户名"
  }
]
}
