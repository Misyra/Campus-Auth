编辑完整的任务配置JSON

JSON 配置说明

支持的步骤类型

  • navigate - 打开网页
  • input - 输入文本
  • click - 点击元素
  • select - 选择下拉框
  • wait - 等待元素出现
  • wait_url - 等待URL匹配
  • eval - 执行JS并保存结果
  • custom_js - 执行JS（无返回值）
  • screenshot - 保存截图
  • sleep - 等待指定时间

条件类型（success_conditions）

  • variable - 变量值判断
  • url_contains - URL包含字符串
  • url_matches - URL正则匹配
  • element_exists - 元素是否存在
  • js_expression - JS表达式判断

可用变量

  • {{username}} - 用户名
  • {{password}} - 密码
  • {{isp}} - 运营商后缀
  • {{url}} - 认证地址

示例

{
"steps": [
  {
    "type": "input",
    "selector": "#username",
    "value": "{{username}}{{isp}}",
    "description": "输入用户名"
  }
]
}
