编辑完整的任务配置JSON

JSON 配置说明

支持的步骤类型

  • navigate - 打开网页（已废弃，请使用 url 字段）
  • input - 输入文本
  • click - 点击元素
  • select - 选择下拉框
  • wait - 等待元素出现
  • wait_url - 等待URL匹配
  • eval - 执行JS（支持返回值存储到变量）
  • screenshot - 保存截图
  • sleep - 等待指定时间
  • ocr - 验证码识别（ddddocr），截图识别图片文字并填入输入框

公共字段

  • frame - 目标 frame 名称或 URL 片段，用于 frameset/iframe 页面

可用变量

  • {{username}} - 用户名
  • {{password}} - 密码
  • {{isp}} - 运营商后缀
  • {{url}} - 认证地址

示例

{
"steps": [
  {
    "type": "input",
    "selector": "#username",
    "value": "{{username}}{{isp}}",
    "description": "输入用户名"
  }
]
}
