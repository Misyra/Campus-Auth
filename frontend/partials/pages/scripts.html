脚本说明

工作原理

Python 脚本在子进程中执行，通过 HTTP 请求直接登录校园网，无需启动浏览器，资源占用极低。

环境变量

  • CAMPUS_USERNAME — 用户名
  • CAMPUS_PASSWORD — 密码
  • CAMPUS_ISP — 运营商（可能为空）
  • CAMPUS_URL — 认证地址

输出说明

脚本只需发送请求，登录是否成功由系统网络检测自动判断，无需脚本自行判断。

  • 脚本正常退出（exit 0）= 执行成功
  • 脚本报错退出（exit 非 0）= 执行失败
  • print() 输出会记录到日志，方便调试

可用 HTTP 库

  • httpx — 已安装，推荐使用，支持 HTTP/2
  • requests — 已安装，最常用的 HTTP 库
  • urllib.request — Python 标准库，无需安装任何依赖

示例（httpx）

#!/usr/bin/env python3
import os, httpx

resp = httpx.post(os.environ["CAMPUS_URL"], data={
    "username": os.environ["CAMPUS_USERNAME"],
    "password": os.environ["CAMPUS_PASSWORD"],
    "operator": os.environ.get("CAMPUS_ISP", ""),
})
print(f"HTTP {resp.status_code}")

示例（urllib 标准库）

#!/usr/bin/env python3
import os, urllib.request, urllib.parse

data = urllib.parse.urlencode({
    "username": os.environ["CAMPUS_USERNAME"],
    "password": os.environ["CAMPUS_PASSWORD"],
}).encode()
req = urllib.request.Request(os.environ["CAMPUS_URL"], data=data)
with urllib.request.urlopen(req, timeout=10) as resp:
    print(f"HTTP {resp.status}")

注意事项

  • 脚本超时默认 60 秒
  • 脚本设为活动任务后，自动监控会使用脚本登录
  • stderr 输出会记录到日志，不影响结果判断
