脚本说明

执行程序

每个脚本可以指定不同的执行程序：

  • Python（默认）— 使用 Python 解释器执行，推荐使用 httpx 或 urllib
  • Shell — Windows 上默认使用 cmd，Linux/macOS 上默认使用 bash，也可自定义 shell 路径
  • 自定义 — 任意可执行文件（.exe、.bat 等）

工作原理

脚本在子进程中执行，通过 HTTP 请求直接登录校园网，无需启动浏览器，资源占用极低。

输出说明

脚本只需发送请求，登录是否成功由系统网络检测自动判断，无需脚本自行判断。

  • 脚本正常退出（exit 0）= 脚本执行无异常
  • 脚本报错退出（exit 非 0）= 脚本执行出错
  • print() 输出会记录到日志，方便调试

示例（Python httpx）

#!/usr/bin/env python3
import httpx

resp = httpx.post("http://10.0.0.1/login", data={
    "username": "your_username",
    "password": "your_password",
    "operator": "cmcc",
})
print(f"HTTP {resp.status_code}")

示例（Shell curl）

#!/bin/bash
curl -s -X POST "http://10.0.0.1/login" \
  -d "username=your_username" \
  -d "password=your_password" \
  -d "operator=cmcc"

注意事项

  • 脚本超时默认 60 秒
  • 脚本设为活动任务后，自动监控会使用脚本登录
  • stderr 输出会记录到日志，不影响结果判断
